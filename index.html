Disk Scheduling Algorithms

Android Studio, Java, OS

I've developed Disk Scheduling Algorithm visualizations in Java, utilizing Android Studio for UI rendering. Leveraging Java's multithreading capabilities, I ensured smooth simulations. The algorithms, FCFS, SSTF, SCAN, C-SCAN, LOOK, and C-LOOK, were implemented with data structures like queues and arrays. These simulations provide an insightful understanding of request handling, optimizing I/O operations, and enhancing overall system performance.
